Word2vec_induction
=============
Use word2vec to find new sentiment words.

Algorithm: 
* get domain text and train word vectors for different domains.
if we have domain text and word vectors,\
    then we can get domain sentiment words.
* use find_new_senti_word_by_word2vecs.py to generate result.
(find_new_senti_word_by_word2vec.py is old version)
